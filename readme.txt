Main code is in swf_termpaper.pml
To run this, you need to install spin. This has been tested on Linux. To get SPIN, go here: 
http://www.spinroot.com/spin/Man/README.html

To run a random simulation path, try this on console:
spin swf_termpaper.pml

To run safety, run the batch file included in this folder: 
./safety 

To run liveness verification, run the batch file included in this folder:
./liveness

Enjoy!
